# 👾 Pokémon TCG Trades

# Sobre o projeto

Pokémon TCG Trades é uma aplicação de trocas cartas desenvolvido com Java + SpringBoot.

Consiste em uma aplicação que realiza além da criação, leitura, atualização e deleção (CRUD) de cartas e usuários, também a troca de cartas entre usuários logados no sistema.

#  🛠️ Tecnologias Utilizadas
## Backend
- Java
- Spring Boot
- JPA / Hibernate
- Maven
- JUnit
- JWT

# ▶️ Como executar o projeto
**Pré-requisitos: Java 21**

## Clonar o repositório
```bash
git clone git@github.com:DyhegoS/pokemontcgtrades.git
```

## Entre na pasta do projeto clonado
```bash
cd pokemontcgtrades
```

## execute o projeto
```bash
./mvnw spring-boot:run
```
